# QR Code Generator

A simple and interactive QR code generator built using HTML, CSS, and Vanilla JavaScript.

## Demo
https://krunxlbhoir.github.io/qr-code-generator/

## How to run locally
1. Download or clone the repository.
2. Open `index.html` in any web browser.

## Features
- Enter any URL or text to generate a QR code.
- Generate QR code instantly with one click.
- Download the QR code as an image.
- Copy the QR code image to clipboard using the Share button.
- Fully responsive across different screen sizes.

## Notes
- Uses the **qrcodejs** library for generating QR codes.  
- Script used:  
  `<script src="https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js"></script>`
- Make sure your browser allows clipboard access for the Share feature.
